Значения и единицы измерения

  • Числа
    • opacity: from 0 to 1
      прозрачность всего элемента, включая содержимое!
      some text
    • lineheight: from 0 to 999
      высота линии относительно высоты текста
      some text
  • Процентные значения

    Ширина желтого бокса указана в процентах,
    которая высчитывается относительно своего родителя - красного контейнера.
    Картинка на бекграунде растянута на 50% по верт и по горизонт.

    Сокращенная запись бекграунда(через пробелы):
    background: color url 50% 50%(size) repeat

    RGB
    RGBA

  • Измерение длины em/rem/ex
    1em = 100% родителя;
    1ex = 1/2em или высоте "x"
    1rem = размер шрифта body
    font_size__px
    если не задан свой размер в px,
    то наследует от родителя(grey_container)     font_size__em
    значение 1.1em     font_size__ex
    эквивалент высоты буквы "x"     font_size__rem
    сбрасывает font_size до уровня < body >
  • calc()
    Ex.= width:calc(100% - 500px);
  • vh, vw, vmin, vmax

    1vw = 1% от ширины окна.
    1vh = 1% от высоты окна.
    1vmin = 1vw или 1vh. выбирает то, которое меньше
    1vmax = 1vw или 1vh. выбирает то, которое больше

    Пример: font-family: Helvetica, sans-serif;

  • font-weight

    Значения: 100,200(lighter),300,400(normal),500,600,700(bold),800,900

  • font-size

    помимо значений и ед измерения, можно задать smaller и larger

  • Свойства font
      font:
    1. font-weight
    2. font-style
    3. font-variant
    4. font-size/line-height
    5. font-family

    Пример записи значений: font: bold italic small-caps 15px/0.5 Arial, sans-serif

  • Добавление внешнего шрифта
    1. В css-файле или < style >: @font-face {src: url(http://..)}
    2. В html файл с помощью link в head(как файл стилей css):
      link rel="stylesheet" href="http://..." type="text/css"
    3. С помощью @import
