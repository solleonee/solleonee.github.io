TRANSITION

  • Transition, Multiple Transitions

    Запись transition:
    transition: transition-property(ex:color) transition-duration(ex:1.2s) transition-timing-function(ex:linear)

    Пример записи в css коде:
    transition: backround-color,color,margin 1.5s linear; - если тайминги и функции одинаковы

    transition:
    background-color 1s linear,
    color 1.7s ease-out,
    margin 1.5s step-end;     - если все разное.

  • Timing function

    Разновидности:
    linear - дефолтный вариант. Действие происходит равномерно во всех стадиях.
    ease - в отличие от linear получает эффект "волны"
    ease-in - в начале медленно, затем быстро
    ease-out - сначала медленно, в середине быстро, в конце медленно
    step-start - выдает сразу окончательный вариант.
    step-end - установленные тайминги считаются в родной позиции, по окончании показывается конечный результат.
    step-end - установленные тайминги считаются в родной позиции, по окончании показывается конечный результат.

TRANSFORM

  • transform

    Пример записи transform:

    transform:
    translate(x[горизонталь],y[вертикаль]);
    scale(x,y);
    rotate(#deg)
    skew(Xdeg,Ydeg)

    Разновидности transform:
    translate - сдвигаем блок по горизонтали и вертикали
    scale - изменяем размер относительно текущего (от 0 до 1, где 1=100%)
    rotate - переворачиваем в градусах по часовой стрелке, с отрицательным значением (-120deg) против часовой стрелки.
    skew - искажаем в градусах по горизонтали и вертикали, могут быть отрицательные значения

  • transform-origin

    Позволяет изменять центр оси, по которой делается transform,задается в процентах
    По дефолту, центр оси находится на 50%ширины и 50%высоты (transform-origin:50% 50%).
    Если по ширине(х)и высоте(у) задано 0% - то центр оси будет в левом верхнем углу, если (x100% y100%) - ось в правом нижнем углу.

    Пример записи:
    transform-origin:50% 50%;

ANIMATION

CSS запись задания анимации:
@keyframes [name] {
from {margin:0;}
20% {margin; 0;}
40% {margin; 0;}
60% {margin; 0;}
to {margin:0;}
}
CSS запись подключения анимации:
div { animation: 0,56s my-custom-name; }
  • animation

    Использование анимации делится на 2 части:
    1. Создание последовательности и имени анимации с помощью правила @keyframes.
    2. Непосредственное использование и параметризация этой анимации.

    animated
  • animation properties
    • animation-name:my-custom-name; - вызов анимации по имени
    • animation-duration:1.65s; - продолжительность анимации
    • animation-interaction-count:3; - количество интераций, бесконечно = infinite
    • animation-direction:reverse|alternate - возможность анимировать в обратную сторону
  • animation and transition

    Чтобы плавно вернуть исходную позицию обьекта после анимации, нужно просто задать анимированному свойству transition у самого обьекта.
