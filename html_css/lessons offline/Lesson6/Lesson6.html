Media-query

Используется при надобности изменять контент взависимости от уловий, например, разрешение монитора, телефона.

  • Варианты подключений

    • < link rel="stylesheet" media="(max-width: 800px)" href="..." /> в < head > html файла
    • В < style > в < head > файла html.
      @media (max-width: 800px) {
      div {display: none;}
      }
      Пример @media. этот текст станет красным при сужении страницы до определенного момента.
