Hyper Text Makeup Language (HTML)

тэги hr, h1(2,3,4,5), p, b/i(strong/em), аттрибуты title

аттрибут title появляется при наведении курсора, пример ниже:

It is our first site

one,
two

тэг а(ссылки)

absolute link
относительная ссылка

Anchors(якоря) - проскроливание текста до определенного места

Используется # в абсолютной ссылке пример https://twitter.com/#news
Используется #+аттрибут id в относительной ссылке пример lesson2.html#news

тэг img(фото), аттрибуты alt,height,width

аттрибут alt - это текст, который прогружается первый или заменяет картинку при битой ссылке
