Позиционирование.

План занятия:
  • position
    • static

      Стандартное позиционирование обьекта(блока) относительно других, соседних.

    • relative
      static
      relative
      static

      Position:relative; + top/right/left: ; смещает блок, не влияя на соседние блоки (bottom:...; не работает).
      При этом может работать margin="..", но он уже будет отталкиваться от границ соседних блоков.

    • absolute
      static
      absolute
      static

      Position: absolute позволяет блоку позиционировать себя относительно всей страницы(body).
      Если задать контейнеру, в котором он находится(родителю), position: relative, тогда absolute будет позиционироваться относительного этого контейнера.
      Margin также прекрасно работают с absolute.

      • Tip1
      • Tip2 + Tip3 (позиционирование по центру родительского блока)

        При position: absolute, а также при установке top,bottom,right,left:0 - выигрывает всегда! top+left.
        Центрирование обьекта(!с указанными width и height!, иначе содержимое растянется по границам родителя) при помощи margin:auto. Соответсвенноб можно только по высоте или ширине.

  • z-index - применяется только к позиционированным элементам(relative,absolute,fixed).
    • z-index(1)

      родитель(контейнер) может отображаться поверх потомка(контента)

    • z-index(2)

      Есть 2 контейнера. В первом(gray) есть потомки(контент-блоки red,blue).
      Потомок с большим z-index перекрывает потомка с меньшим(red > blue).
      Родитель с большим z-index перекрывает родителя с меньшим (GRAY > GREEN), при одинаковых значениях поверх будет тот элемент, который будет ниже в коде HTML.
      А ТАКЖЕ ПОТОМКИ РОДИТЕЛЯ С БОЛЬШИМ z-index ПЕРЕКРЫВАЮТ ВТОРОГО РОДИТЕЛЯ(GRAY+red,blue >>> GREEN)
