Коды состояния HTTP

  • Информационные ответы(1хх)
    • 100 Continue
    • 101 Switching protocols
    • 102 Progressing
  • Ответа успеха(2xx)
    • 200 OK
    • 201 Created
    • 202 Accepted
  • Ответы перенаправления(3xx)
    • 301 Moved Permanently
    • 302 Moved Temporarily
    • 304 Not Modified
  • Ошибки клиента(4xx)
    • 400 Bad Request
    • 403 Forbidden
    • 404 Not Found
    • 408 Request Timeout
  • Ошибки сервера(5xx)
    • 500 Internal Server Error
    • 502 Bad Gateway
    • 504 Gateway Timeout
    • 503 Service Unavailable

Аттрибуты cookie

Пример: yandexuid=897097897897897; Expires=Sat, 08 Jan 2022 09:22:23 GMT path=/; domain=.yandex.ru
